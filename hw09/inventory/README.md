### Сервис «Склад»

**Описание**

Сервис предоставляет API для работы со складом

**Запросы**

- None

**Команды**

- `POST /reserveGoods` — бронирование товаров
- `POST /cancelGoodsReservation` — отмена бронирования товаров (компенсация для метода бронирования, используется для саги)

**События**

- None

**Зависимости**

- None

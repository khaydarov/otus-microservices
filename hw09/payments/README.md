### Сервис «Биллинг»

**Описание**

Сервис предоставляет API для работы с платежами

**Запросы**

- None

**Команды**

- `POST /makePayment` — оплата заказа
- `POST /cancelPayment` — отмена оплаты заказа (компенсация для метода оплаты, используется для саги)

**События**

- None

**Зависимости**

- None
